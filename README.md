## Bhakti Jain Portfolio

This is the source code for the personal portfolio website of **Bhakti Jain**.

The site is built with:

- Vite
- React
- TypeScript
- shadcn-ui
- Tailwind CSS

### Getting started

1. Install dependencies:

```sh
npm install
```

2. Run the development server:

```sh
npm run dev
```

3. Create a production build:

```sh
npm run build
```

4. Preview the production build:

```sh
npm run preview
```

You can customize the content and sections in the `src` directory to keep the portfolio up to date with projects, experience, and skills.
